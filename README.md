# Student Grade Manager 📊

A Python command-line tool that takes student marks, calculates average and assigns grades automatically.

## Features
- Takes marks for 4 subjects
- Calculates average automatically
- Assigns grades from A+ to F
- Gives performance feedback

## How to Run
1. Make sure Python is installed
2. Run the file:
```
python grades.py
```

## Sample Output
```
Student: Pratham
Average: 85.0
Grade: A | Excellent!
```

## Technologies Used
- Python 3.x
- Core concepts: variables, user input, arithmetic, conditional logic
